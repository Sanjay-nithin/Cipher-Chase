**#CIPHER CHASE**
To run this game follow the upcoming steps:
Step 1: Open a terminal in your code editor (ex: Vs code)
Step 2: Run the command **"python manage.py runserver"**
Step 3: Since this runs directly by the link we have given. This will first show a page not found error. In the address bar add "http://127.0.0.1:8000**/play**"
Step 4: Now you can enjoy your game.
