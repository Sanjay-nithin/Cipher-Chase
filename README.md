How to Run the Game
Follow these steps to start the game in your local development environment:

Step 1: Open a Terminal
Open a terminal in your code editor (e.g., VS Code).
Step 2: Start the Django Server
Run the following command in the terminal:
**python manage.py runserver**
Step 3: Access the Game
By default, the server runs at http://127.0.0.1:8000/.
This may show a Page Not Found (404) error.
Step 4: Navigate to the Game Page
In the address bar, manually add /play to the URL:
**http://127.0.0.1:8000/play**
Press Enter to load the game.
Now, enjoy playing Cipher Chase! 🚀
